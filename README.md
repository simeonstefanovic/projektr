# MATAN Analiza

Analiza prolaznosti MA1 i MA2 na FER-u (2018-2024).

## Preduvjeti

- Python 3.10+

## Setup

**Windows:**
```
setup.bat
```

**Linux/Mac:**
```
chmod +x setup.sh
./setup.sh
```

## Pokretanje

**Windows:**
```
run.bat
```

**Linux/Mac:**
```
./run.sh
```

Rezultati idu u `output/` folder - statistike i grafovi.

## Dodavanje novih podataka

Baci CSV u `data/MATAN/` s imenom tipa `MA1_2025_clean.csv` ili `MA2_2025_clean.csv`. Program automatski pokupi sve.
